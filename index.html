Escopo

GLOBAL - variáveis declaradas FORA de uma função, podem ser chamadas novamente no código.

LOCAL - variáveis declaradas DENTRO de uma função, usadas somente dentro da função.

Variáveis e Constantes

Dados guardados na memória ram que são usados nas taferas.

  • VAR - tipo de variável que ultimamente gera problemas na memória.
  • LET - criada para resolver os problemas da 'Var'. Pode ter seu valor redefinido.
  • CONST - utilizada para definir dados constantes. Não pode ter seu valor redefinido.

Declarando Variáveis

LET nomeDaVariavel = Dados

let nome = `Jésus`

Const ano = 1998

Variáveis podem ser declaradas usando $ e _ para diferenciar variáveis

let nome = `Jésus`

let $nome = `Jésus`

let _nome = `Jésus`

Operaodres Lógicos

Usam valores booleanos (true e false) para fazer operaçoes lógicas.

ex : comparação entre valores

&& (and, e)

if (variavel A = Valor Booleano A) && (variavel B = Valor Booleano B){}

retorna verdadeiro quando os dois valores declarados são verdadeios.

Caso contrario retorna falso.

  • A + B = AB
  • T + T = TRUE
  • T + F = FALSE
  • F + T = FALSE
  • F + F = FALSE

|| (or, ou)

if (variavel A = Valor Booleano A) || (variavel B = Valor Booleano B){}

Reotnra veirdadiero quando uma das informações são verdadeiras.

Retorna falso qunado as duas são falsas.
